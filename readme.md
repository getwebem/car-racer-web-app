#Car Race app
###**Getting started**
###Technologies Used
####HTML5, CSS3, jQuery 2.1.4, Photoshop
 - Run :  `git clone https://github.com/getwebem/car-racer-web-app.git`
 - Run :  `cd car-racer-web-app`
 - Run :  `Brackets .`
 - Open:  `Index.html`
 - Open:  `Live Preview`
